# Java-Love-Test

This project, Java Love Test, is a love quiz developed entirely in Java, utilizing JFrames, Interfaces, and Object-Oriented Programming (OOP). The aim of the quiz is to provide users with an interactive and fun experience as they explore romantic compatibility.

## Key Features

- **Timed Questionnaire:** The questionnaire consists of a series of timed questions that users must answer within a predetermined time limit.
- **Graphical User Interface (GUI):** The application utilizes JFrames to offer a friendly and intuitive graphical interface.
- **Results Analysis:** Upon completion of the questionnaire, a percentage is calculated representing the likelihood of finding true love based on the user's responses.

## Requirements

- Java Development Kit (JDK)
- Internet Connection (for downloading dependencies if necessary)

## Usage Instructions

1. Clone or download the repository to your local machine.
2. Open the project in your preferred Java development environment.
3. Compile and run the project.
4. Follow the on-screen instructions to complete the questionnaire.
5. Upon completion, you will receive your love compatibility percentage.
